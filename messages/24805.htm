>>2.1 Rechtschreibfehler
>>Ich bin kein Germanistikprofessor. Aber ich erwarte schon, das man ein Posting wenigstens halbwegs entziffern kann. Dazu gehört auch eine Groß-/Kleinschreibung. Richtige Typos kann man verzeihen, da sie jedem passieren. Aber wie gesagt: Achtet auf die Struktur, auf euere Groß-/Kleinschreibung und auf die Satzzeichen. Ich kann mir nicht helfen, aber wenn ein Posting zu viele Fehler enthält klicke ich ganz schnell weiter, weil mir die Dechiffrierung von rechtschreibschwachem Gefasel zu mühselig ist.
>>2.2 Dummer Titel
>>Ich lese zwar alles, aber schaue trotzdem auf den Betreff (oder das Subject). Ich nenn mein Posting ja auch nicht "Frosch sucht Föhn" sondern ganz einfach nach dem Inhalt! So schwer? >>2.3 Thema
>>Ja, das ist ein Forum zum Thema Warhammer 40,k. Also gehe ich davon aus, das es auch um 40,k geht. Setzt ein (OT = Off Topic = Nicht zum Topic gehörend) vor ein Posting das keinen Inhalt zum Thema besitzt. Natürlich soll das kein Freibrief zum Trashposten sein, aber ich denke, dieses Posting ist ein gutes Beispiel für diesen (OT)-Zusatz. Wenn ihr nix zum Thema zu sagen habt, lasst es oder schreibt eine Mail. Das gleiche gilt auch für das (NT = No Text, alternativ KT = Kein Text). Wenn ihr nur einen kurzen Satz posten wollt, macht es doch bitte so.
